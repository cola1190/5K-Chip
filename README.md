# 5K-Chip

This repository contains scripts and analysis workflows related to the development of a pear 5K liquid-phase functional SNP chip.

## Project overview

This project was developed using whole-genome resequencing data from 668 pear accessions together with agronomic trait data collected across multiple years. Genome-wide association studies (GWAS) were conducted to identify loci associated with important fruit and breeding-related traits, and high-confidence, high-signal SNPs were subsequently prioritized and integrated to develop a breeding-oriented 5K liquid-phase SNP chip for pear.
The project is intended to support marker-assisted analysis, trait-associated SNP discovery, and molecular breeding applications in pear.

## Repository contents

This repository currently includes scripts or workflows related to:

- `SNP-Calling`
- `SNP-filter`
- `SNP-genotype`
- `PCA-analysis`
- `Admixture`
- `Fst`
- `GWAS-analysis`
- `LDBlockShow`
- `SNP-density-cmplot`
- `phenotype-gene-associate`
- `Circle-chip-snp.R`

## Current release status

This repository does not yet contain the complete set of scripts used in the full project. In particular, scripts related to genomic selection (GS) and some downstream prediction analyses are not included in the current release.

These materials are currently being cleaned, annotated, and standardized, and will be uploaded after formal publication of the associated manuscript.

## Notes

The current version of this repository is intended to document the major analysis framework used for SNP discovery, population analysis, and GWAS-based marker selection in the pear 5K chip project. Additional scripts, documentation, and updates will be added in future releases.
